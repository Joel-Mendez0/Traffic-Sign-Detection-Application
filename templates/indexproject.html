Road Sign Object Detection Demo

Welcome to the Road Sign Object Detection project! This initiative is dedicated to enhancing road safety by leveraging cutting-edge technology for the real-time detection and analysis of road signs. Using the YOLOv8 model, a state-of-the-art object detection algorithm, the project identifies and locates various road signs in live video streams or images.
